# Figures

This folder contains the final plots featured in the paper.
All figures were generated using the scripts available in `scripts/` with the parameters noted in each file.

These scripts produce the following figures:

ZVCFigures.py → ZVCAtL4.png, ZVCAtL1Closed.png, ZVCAtL1Opened.png

L1NeckTransitFigures.py → ReflectionAtL1.png, TransitAtL1.png

HorseshoeAndTadpoleFigures.py → SmoothTadpole.png, SmoothHorseshoe.png

ChaoticHorseshoeAndTadpoleFigures.py → ChaoticTadpole.png, ChaoticHorseshoeWithZVCs.png
